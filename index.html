Jungle-Object-Web by DuncanCragg

Jungle is a Java application server for building a corner of a global interactive object Web. You write your application in declarative style rules - in native Java. It is inherently concurrent and easily distributed.

Jungle is a Java application server for building a corner of a global interactive object Web. You write your application in declarative style rules - in native Java. It is inherently concurrent and easily distributed. It is the Java Reference Implementation of FOREST: http://forest-object-web.org/
